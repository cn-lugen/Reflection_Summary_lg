# 导数定义
导数的定义为：

f'(x) = lim (h -> 0) [f(x+h) - f(x)] / h

其中，f'(x) 表示函数 f(x) 在点 x 处的导数，h 是一个趋近于 0 的增量。


# 四则运算
- (u+v)'=u'+v'
- (u-v)'=u'-v'
- (uv)'=u'v+uv'
- (u/v)'=(u'v-uv')/v^2

# 复合函数的运算法则

# 函数的可导性与连续性之间的关系
- Th1 函数f(x)在x0处可微=在x0处可导
- Th2 若函数f(x)在点x0处可导，则在点处x0连续，反之则不成立。即函数连续不一定可导。(例如: y=|x|)
